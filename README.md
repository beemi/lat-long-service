# lat-long-service

Get Postcode details using https://expressjs.com/

## routes

Get longitude and latitude from postcode

```shell
curl -X GET http://localhost:7000?postcode=SW1A 1AA
```

```shell
cur -X GET http://localhost:7000/postcode/SW1A 1AA
```
